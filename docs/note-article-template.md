# 「若くして高年収」が分かる、上場企業ランキングを作りました

## 一行サマリ

東証プライム / スタンダード / グロースに上場している全企業の有価証券報告書（EDINET）から
**平均年収・平均年齢・平均勤続年数** を取得し、「若くして高年収」を **業種補正されたスコア** で
ランキング表示できるサイトを公開しました。

🔗 **<YOUR_SITE_URL>**

## なぜ作ったか

- 転職検討時に「年収だけ」「平均年齢だけ」を見ても、業種ごとの構造の違いで比較しづらい
- 既存の年収ランキングサイトは広告まみれ・データが古い・業種補正がない
- 有価証券報告書という **公開情報** が EDINET API でちゃんと取れる時代なので、自分で作ってみたかった

## 何ができるか

### 1. ランキング & 検索 & フィルタ

- 全 ${COUNT} 社（順次拡大）を、年収 / 平均年齢 / 独自スコア / 5年成長率 などでソート
- 業種・市場・年収レンジ・年齢レンジ・従業員数で絞り込み
- 4,000社の仮想スクロールで瞬時表示、URL を共有すると同じフィルタ条件で開ける

### 2. 独自スコア「若くして高年収」指標

```
スコア = (平均年収 / 業界平均年収) × (業界平均年齢 / 平均年齢) × 50
```

- 業界平均なら 50、業界の2倍年収なら 100
- 銀行 vs IT のような業種差を吸収
- S / A / B / C / D の5段階グレードを付与

スクリーンショット例 → **${COMPANY_EXAMPLE}** は ${COMPANY_GRADE} グレード（${COMPANY_SCORE}点）

### 3. 年収シミュレーション

平均値からあなたの年齢の推定年収を試算:

- 5年前 / 現在 / 5年後 / 10年後 / 22-65歳の生涯年収
- 業種別の昇給カーブ（年功型 / バランス型 / 成果型）を反映
- 入力した年齢にマーカーが立つインタラクティブグラフ

### 4. 最大4社の横並び比較

- レーダーチャートで「年収」「若さ」「スコア」「規模」「勤続」を多角比較
- 比較結果をURL共有可能

### 5. その他

- 業種別ランキング（東証33業種）
- お気に入り保存（localStorage）
- ダーク / ライト モード対応
- スマホでも快適に読めるレスポンシブ
- 各企業ページに OGP画像を自動生成、SNS共有時に綺麗に表示

## 使い方

1. <YOUR_SITE_URL> にアクセス
2. 上部の検索バーに企業名 or 証券コードを入力
3. 各企業をクリックして詳細・スコア・シミュレーションを表示
4. ★アイコンでお気に入り、レーダーで比較

## 技術的なポイント（興味のある方向け）

完全無料枠 + 個人開発に最適な構成:

| Layer | 採用 | コスト |
|---|---|---|
| Frontend | Next.js 15 (App Router) on Vercel Hobby | $0 |
| DB | Supabase Postgres (Free 500MB) | $0 |
| Data Pipeline | Python 3.12 + lxml (XBRL直接パース) | $0 |
| Cron | GitHub Actions (年1回) | $0 |
| OGP | @vercel/og (Edge Runtime) | $0 |
| **合計** | | **$0 / 月** |

データ更新は **毎年7月1日 00:00 JST** に GitHub Actions が自動実行。
3月期決算企業の有報がほぼ出揃うタイミングで一気に取り込みます。

## 注意事項

- 数値はすべて EDINET 開示情報を基にした **推計値**。実際の給与を保証するものではありません
- 年収シミュレーションは業種カーブに当てはめた目安で、個人の役職・成果・転職歴により大きく変動します
- 個別企業の異議申し立てはサイト内のお問い合わせフォームから受け付けます
