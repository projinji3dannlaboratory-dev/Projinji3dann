# 年収シミュレーション設計

## 目的

平均年齢 $A_c$・平均年収 $S_c$ から、その企業に在籍した場合の年齢別年収カーブを推計し、過去・将来・生涯年収を試算する。

## モデル

業種ごとに「年収 = ベース × カーブ係数」を仮定。
カーブ係数 $f(\text{age})$ は経年とともに緩やかに成長し、ピーク後に緩やかに低下するロジスティック型。

```
f(age) = base + (peak - base) / (1 + exp(-k * (age - mid)))
```

- `base`: 22歳時点の係数（業種別、例: 0.55）
- `peak`: 50〜55歳のピーク係数（例: 1.45）
- `k`: 成長の急峻さ（業種別、例: 0.18）
- `mid`: 成長の中央値となる年齢（業種別、例: 38）

業種別パラメータは `lib/salary-curve-params.ts` に定義。
当面はざっくり3グループ（年功型 / バランス型 / 成果型）で運用。

## 校正

企業ごとに、平均年齢 $A_c$ における係数が「平均給与に一致するように」スケールする:

```
scale = S_c / f(A_c)
estimated_salary(age) = scale * f(age)
```

これにより、業種カーブの「形」は維持しつつ、企業固有の水準にフィットする。

## 出力

ユーザーが現在年齢 $a_0$ を入力した場合:

- **5年前の想定年収** = `estimated_salary(a_0 - 5)`
- **現在の想定年収** = `estimated_salary(a_0)`
- **5年後** = `estimated_salary(a_0 + 5)`
- **10年後** = `estimated_salary(a_0 + 10)`
- **生涯年収（22-65歳）** = `Σ estimated_salary(age) for age in 22..65`

## 表示上の注意

- 「あくまで開示値からの推計であり、個人の役職・成果・転職状況により大きく変動します」
- グラフは入力年齢を強調表示
- 業種カーブのタイプ（年功型/成果型）も併記

## 業種カーブパラメータ（仮）

| グループ | 該当業種例 | base | peak | k   | mid |
|---|---|---|---|---|---|
| 年功型 | 銀行・電力・建設・公益 | 0.50 | 1.55 | 0.16 | 40 |
| バランス型 | 製造・化学・小売・食品 | 0.55 | 1.45 | 0.18 | 38 |
| 成果型 | 情報通信・証券・コンサル | 0.60 | 1.40 | 0.22 | 35 |

将来的に各業種の実績データから自動キャリブレーションする。
