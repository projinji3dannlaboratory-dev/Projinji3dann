# 独自スコアの設計

## 目的

「若くして高年収を実現している企業」を、業種間の年収・年齢構造の差を補正したうえで定量化する。

## 入力

| 変数 | 記号 | 出典 |
|---|---|---|
| 企業の平均年間給与 | $S_c$ | 有価証券報告書「従業員の状況」 |
| 企業の平均年齢 | $A_c$ | 同上 |
| 業種の平均年間給与 | $S_i$ | 同業種企業の平均（事前集計） |
| 業種の平均年齢 | $A_i$ | 同上 |
| 業種内の標準偏差 | $\sigma^S_i, \sigma^A_i$ | 偏差値計算用 |

## 計算

### 1. 業種内偏差値

```
salary_deviation = 50 + 10 * (S_c - S_i) / σ^S_i        (年収は高いほど高い)
age_deviation    = 50 + 10 * (A_i - A_c) / σ^A_i        (年齢は若いほど高い)
```

### 2. 独自スコア (raw_score)

```
raw_score = (S_c / S_i) * (A_i / A_c) * 50
```

- 業界平均そのものなら $1.0 \times 1.0 \times 50 = 50$
- 「年収2倍 × 年齢同等」なら 100
- 「年収同等 × 年齢25%若い」なら ~67

スコアは 50 を平均、 100 を「業界平均の2倍年収または2倍若い」相当として直感的に解釈できる。

### 3. グレード

| Grade | 範囲 | 意味 |
|---|---|---|
| **S** | raw_score ≥ 80 | 業種内トップクラス。年収高 × 平均年齢若の典型 |
| **A** | 65 ≤ raw_score < 80 | 業種内上位 |
| **B** | 50 ≤ raw_score < 65 | 業種内平均以上 |
| **C** | 35 ≤ raw_score < 50 | 業種内平均未満 |
| **D** | raw_score < 35 | 業種内下位 |

### 4. 業界補正係数

`industry_correction = (S_c / S_i) * (A_i / A_c)` をそのまま記録。
ユーザー画面では「業界平均の何倍働きやすい数値か」として表示。

## 注意

- 単体（単独決算）ベースの開示値を採用
- 平均年齢・平均給与どちらかが欠損する企業はスコア計算対象外（grade=null）
- 業種内サンプルが10社未満の場合は警告フラグを立てる（信頼性が低いため）
- 50% を切るような極端な少サンプル業種は将来的に親業種に統合する余地あり
